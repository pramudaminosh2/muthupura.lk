🎓 KEY TECHNICAL DECISIONS
───────────────────────────────────────────────────────────────────

1. Use CloudinaryStorage
   → Automatic upload without manual API calls
   → Simpler code
   → Built-in error handling

2. Store Full URLs
   → Portable (can switch CDN anytime)
   → Works offline from cache
   → No URL construction needed

3. JSON Array Format
   → Flexible for any number of images
   → Easy to parse
   → Standards-compliant

4. Primary Image Field
   → Backward compatibility
   → Quick access to main image
   → Fallback value available

5. Environment Variables
   → Security best practice
   → Easy credential rotation
   → Different values per environment
